Hi, I'm **Eljames Benitez** — a Full Stack Web Developer passionate about building clean, functional web applications. This portfolio highlights my work, tech stack, and how to get in touch.

- **Backend:** PHP 8.5, Laravel 13
- **Frontend:** Blade Templates, HTML5, CSS3, JavaScript
- **Database:** MySQL
- **Version Control:** Git
- **Other:** Vue.js, Tailwind CSS

## Getting Started

`ash
# Clone the repository
git clone https://github.com/elj9mes-max/portfolio.git
cd portfolio

# Install dependencies
composer install
npm install

# Set up environment
cp .env.example .env
php artisan key:generate

# Run the app
php artisan serve
npm run dev
`

Then visit `http://localhost:8000`.
